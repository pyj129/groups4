# 🔢 소인수분해 탐구 학습실

중학교 1학년 수학 **소인수분해** 단원을 위한 인터랙티브 웹 학습 앱입니다.
학생들이 자연수를 입력하여 소인수분해 과정을 **3가지 방법**으로 시각화하고, 
약수를 자동 생성한 후 **퀴즈를 통해 자기주도적으로 학습**을 점검할 수 있습니다.

## ✨ 핵심 기능

### 📋 기능 1: 소인수분해 시각화 및 유일성 탐색
- **입력 기능**: 학생이 원하는 자연수 입력 가능
- **시각화 (3가지 방법)**:
  1. **나열법**: 자연수를 다른 자연수의 곱으로 계속 나누어 소수들만의 곱으로 표현
  2. **소인수분해 가지치기**: 트리 구조로 소인수분해 과정 시각화
  3. **L자형 나눗셈**: 가장 작은 소수부터 차례대로 아래로 나누는 전통적 방식
- **결과 출력**: 거듭제곱 형태로 정리 (예: 2² × 3 × 5)

### 📊 기능 2: 소인수분해를 이용한 약수 자동 생성
- **약수 표 생성**: 교과서 스타일의 약수 구하기 표 자동 생성
- **약수 리스트 및 개수**: 구해진 약수의 전체 리스트와 총 개수 표시
- **생각해보기 발문**: "수가 크다고 해서 항상 약수의 개수가 많을까?" 
  - 예시: 24(약수 8개) vs 25(약수 3개)를 시각적으로 비교

### 🎯 기능 3: 학습 점검용 실시간 퀴즈
- **퀴즈 유형** (3가지):
  1. 소인수분해 하기
  2. 거듭제곱으로 올바르게 나타내기
  3. 약수의 개수 구하기
- **인터랙션**: 학생이 정답 입력 → 즉시 정오답 확인 → 틀리기 쉬운 부분 힌트 제공

## 💡 주요 알고리즘

### 소인수분해 함수
```python
def prime_factorization(n: int) -> List[int]:
    """자연수를 소인수분해하여 소인수 리스트 반환"""
    factors = []
    d = 2
    while d * d <= n:
        while n % d == 0:
            factors.append(d)
            n //= d
        d += 1
    if n > 1:
        factors.append(n)
    return factors
```

### 약수 자동 생성
각 소인수의 지수를 0 이상으로 조합하여 모든 약수를 생성합니다.
약수의 개수 = (지수₁+1) × (지수₂+1) × ... × (지수ₙ+1)
